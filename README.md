Young Invest — Investment Portal
​This repository contains the frontend application for Young Invest, a direct-funding portal designed for early-stage builders and founders.  
​Overview
​Young Invest is built for founders who value direct execution over traditional, slow-moving investment processes. The portal allows builders to apply for funding directly without the need for warm introductions or formal pitch decks.  
​Features
​Five-Step Application Flow: A guided wizard that collects essential project information from founders.  
​Flexible Funding Models: Supports both Equity (share of company) and Revenue Share (pay from earnings) options.  
​Integrated Review Process: Allows founders to review their application details before final submission to ensure data accuracy.  
​Direct Contact Channels: Provides immediate post-submission links for WhatsApp, Telegram, and Zangi for streamlined communication.  
​How to Deploy
​To use this as your live portal:
​Configure the Script: Ensure the SCRIPT_URL variable in the young-invest (1).html file is updated with your Google Apps Script Web App URL to receive application data.  
​Enable GitHub Pages:
​Go to your repository Settings.
​Navigate to Pages.
​Under Build and deployment, set the source to your main branch.
​Once enabled, your site will be live at a public URL provided by GitHub.
​Disclaimer
​Submitting an application does not guarantee funding. All terms, whether equity or revenue-share, are agreed upon personally between the investor and the founder before any capital changes hands.
